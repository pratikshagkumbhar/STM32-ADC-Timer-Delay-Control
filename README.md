# STM32-ADC-Timer-Delay-Control
STM32-based embedded system that interfaces ADC and TIM11 to generate voltage-dependent delays (5s, 10s, and 15s). Developed using STM32CubeIDE and HAL drivers with UART monitoring and GPIO control.

## Features
- ADC voltage sensing using PA0
- TIM11-based delay generation
- UART monitoring of ADC values
- LED control using GPIO
